# estacionamiento# 🚗 Sistema de Estacionamiento (React + Node.js)

Sistema de gestión para un estacionamiento desarrollado con **React (Frontend)** y **Node.js + SQLite (Backend)**.  
Permite el control de autos que ingresan y egresan, cálculo de tarifas en base al tiempo estacionado y configuración dinámica de precios.
